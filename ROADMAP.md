# 🚀 Zyloq AI Engineering Academy Roadmap

This roadmap defines the competency-based learning journey for engineers joining Zyloq AI.

## Learning Phases

### Phase 1 — Foundations
**Objective:** Understand computers from first principles.

Topics:
- Computer Architecture
- CPU, GPU, RAM
- Storage
- Networking
- Operating Systems
- Linux & Ubuntu
- Virtual Machines
- Cloud Fundamentals

**Outcome**
- Build and understand a Linux development environment.

---

### Phase 2 — Development Environment
Topics:
- Git
- GitHub
- GitHub CLI
- SSH
- VS Code
- Bash
- tmux
- Package Managers

**Outcome**
- Build a professional engineering workstation.

---

### Phase 3 — Programming
Topics:
- Python
- Node.js
- Virtual Environments
- Testing
- Debugging
- Logging
- FastAPI

**Outcome**
- Develop production-quality backend services.

---

### Phase 4 — Cloud & Infrastructure
Azure, Docker, Kubernetes, CI/CD, Monitoring, Security.

### Phase 5 — Machine Learning
Statistics, Regression, Classification, Feature Engineering.

### Phase 6 — Deep Learning
Neural Networks, CNNs, RNNs, LSTMs, PyTorch.

### Phase 7 — Large Language Models
Transformers, Attention, Embeddings, Tokenization, Fine-tuning, RLHF, Inference.

### Phase 8 — AI Systems Engineering
LangChain, LangGraph, RAG, LiteLLM, MCP, OpenRouter, AI Agents.

### Phase 9 — Production AI
vLLM, Ollama, Scaling, Observability, Cost Optimization.

### Phase 10 — Zyloq AI Engineering
Architecture, Standards, Reviews, Documentation, Product Engineering.
